XHdcp22_Tx_PairingInfo XHdcp22_Tx_PairingInfoStore[XHDCP22_TX_MAX_STORED_PAIRINGINFO]

#include <xhdcp22_tx.c>

This array contains the pairing info keys that are used for authentication with stored Km.

It uses a mechanism that tries to find an empty entry for adding a new one.

  • Optionally move to non-volatile secure storage to have persistent stored Km authentication even after power-up.
  • It is not required by the protocol, but InvalidatePairingInfo()could be used if a failure occurs in stored Km authentication, so automatically the non-stored Km authentication is executed on next authentication.
  • If no empty slots, and the array is full, the first entry is returned for pairing info storage.
